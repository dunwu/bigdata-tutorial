      # Kafka 教程

      Kafka 是 Apache 的开源项目。Kafka 既可以作为一个消息队列中间件，也可以作为一个分布式流处理平台。

      Kafka 用于构建实时数据管道和流应用。它具有水平可伸缩性，容错性，快速快速性。
